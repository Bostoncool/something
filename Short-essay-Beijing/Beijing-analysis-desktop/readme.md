# 北京空气质量数据分析脚本说明

本文件夹包含用于分析北京空气质量数据的Python脚本，主要功能包括数据读取、清洗、统计分析及可视化。

## 脚本功能简介

### 1. 污染物超标分析
- **`AQI_Over_200.py`**: 
  - **功能**: 分析 **AQI** 超标（>200，重度污染）情况。
  - **输出**: 生成年度、月度、季节的超标发生率统计图表，以及各监测站点的超标分布饼图。
- **`AQI-MAX.py`**: 
  - **功能**: 类似于 `AQI_Over_200.py`，但代码主要针对 **NO2** 进行筛选分析（阈值设为 >1200）。
  - **输出**: NO2 高浓度情况的年度、月度、季节趋势及站点分布图。
- **`Pollutants.py`**: 
  - **功能**: 专门分析 **臭氧 (O3_24h)** 超标（>265）情况。
  - **输出**: O3 超标天数的统计及可视化报告（年度、月度、季节及站点分布）。

### 2. 时间序列趋势分析
- **`Date.py`**: 
  - **功能**: 处理特定年份（如2024年）的 **PM2.5, PM10, AQI** 数据。
  - **输出**: 绘制全站点平均及各监测站点的每日污染物浓度变化趋势图。
- **`Date_Separate.py`**: 
  - **功能**: 批量处理 **2015-2024年** 的 **PM2.5, PM10, AQI** 数据。
  - **输出**: 分年份生成主要污染物的每日趋势图（包含全站点平均及各分站点）。
- **`Date_Separate_other_4.py`**: 
  - **功能**: 批量处理 **2015-2024年** 的 **CO, NO2, O3, SO2** 数据。
  - **输出**: 分年份生成其他四种气体污染物的每日趋势图。

### 3. 长期统计与季节分析
- **`Benchmark.py`**: 
  - **功能**: 计算 **PM2.5, PM10, AQI** 的 **年平均浓度**。
  - **输出**: 计算年际变化率，绘制年度平均污染水平折线图，并保存各站点的年均值数据。
- **`Season_separte.py`**: 
  - **功能**: 分析 **CO, NO2, SO2, O3** 的 **季节性变化**。
  - **输出**: 绘制季节变化趋势图（折线图）和热力图，展示不同年份四季的污染物浓度变化，并保存站点季节统计数据。

## 注意事项
- 脚本中通常硬编码了数据源路径（如 `C:\Users\IU\Desktop\...`），使用前请根据实际情况修改 `root_folder` 或 `folder_path` 变量。
- 输出文件默认保存在用户桌面的特定文件夹中。
