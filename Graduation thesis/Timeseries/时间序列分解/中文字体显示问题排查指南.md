# Matplotlib 中文字体显示问题排查指南

> 本文档汇总了三大城市群 STL 分析脚本中，图表中文显示为方框的问题成因、脚本差异、解决方案及验证方法，便于日后遇到类似问题时快速排查与修复。

---

## 一、问题现象

- **正常表现**：图表中的中文标题、坐标轴标签、图例等正确显示为汉字。
- **异常表现**：中文被渲染为小方框（□），即字体缺失时的占位符。

---

## 二、中文字体报错 / 方框的常见原因

### 2.1 根本原因

Matplotlib 默认使用 **DejaVu Sans** 等西文字体，**不支持中文字符**。当未正确指定中文字体时，系统会回退到这些字体，导致中文无法渲染，显示为方框。

### 2.2 具体诱因

| 诱因 | 说明 |
|------|------|
| 未设置中文字体 | `rcParams` 中未配置支持中文的字体 |
| 字体链被覆盖 | 使用 `sns.set_theme()` 等操作后，主题可能重置 `font.sans-serif` |
| 自动选择失败 | 动态探测字体时，若未检测到可用中文字体，回退到 `DejaVu Sans` |
| 输出格式差异 | PNG 在保存时即栅格化，字体缺失会直接变成方框；SVG 为矢量，部分查看器可能做字体替换，表现可能不同 |

## 三、不同脚本之间的差异

以 **三大城市群季节PM2.5时序（新）.py**（正常显示）与 **三大城市群（月）STL.py**（曾出现方框）为例：

| 对比项 | 正常脚本 | 异常脚本（修改前） |

| 配置逻辑 | 固定字体链，直接设置 | 自动探测可用字体，选一个 |

### 3.3 输出格式

| 对比项 | 正常脚本 | 异常脚本（修改前） |
|--------|----------|--------------------|

## 四、导致变成方框的原因（汇总）

1. **字体链中优先使用了不支持中文的字体**（如 `DejaVu Sans`）作为首选或回退。
2. **自动探测失败**：本机未安装预设的中文字体，或探测逻辑有误，导致选中了西文字体。
3. **Seaborn 主题覆盖**：`sns.set_theme()` 在设置字体之后执行，或通过 `rc=plt.rcParams` 传入时，主题内部可能重置 `font.sans-serif`。
4. **PNG 输出**：保存为 PNG 时，字体信息在生成时即被栅格化，一旦字体错误就无法补救。

---

## 五、不变成方框的原因（正常脚本为何正常）

1. **固定中文字体链**：明确使用 `SimHei`、`Microsoft YaHei` 等支持中文的字体，不依赖自动探测。
2. **无 Seaborn 主题覆盖**：仅设置 `mpl.rcParams`，不调用会重置字体的 theme。
3. **SVG 输出**：矢量格式对字体依赖更友好，部分场景下即使配置略有偏差，查看器也可能做字体替换。

## 六、修改方案（推荐做法）

### 6.1 字体配置（核心）

```python
import matplotlib as mpl
import matplotlib.pyplot as plt
import seaborn as sns

# 1. 先设置 seaborn 主题（若有使用）
sns.set_theme(style="whitegrid")

# 2. 再强制覆盖字体相关 rc，避免被 theme 重置
chinese_font_chain = [
    "SimHei",
    "Microsoft YaHei",
    "Arial Unicode MS",
    "SimSun",
    "Noto Sans CJK SC",
    "Source Han Sans SC",
]
mpl.rcParams["font.family"] = "sans-serif"
mpl.rcParams["font.sans-serif"] = chinese_font_chain
mpl.rcParams["axes.unicode_minus"] = False  # 负号显示正常
```

**要点**：
- 使用**固定字体链**，不要用 `DejaVu Sans` 作为首选。
- 若使用 seaborn，**先** `sns.set_theme()`，**再**设置字体 rc。
- `axes.unicode_minus = False` 可避免负号显示异常。

### 6.2 同时输出 PNG 与 SVG（论文推荐）

```python
def save_figure_dual(fig, save_path_png: str, dpi: int = 300) -> None:
    """同时保存 PNG 与 SVG 版本。"""
    fig.savefig(save_path_png, dpi=dpi, bbox_inches="tight")
    save_path_svg = os.path.splitext(save_path_png)[0] + ".svg"
    fig.savefig(save_path_svg, format="svg", bbox_inches="tight")
```

- **PNG**：便于预览、插入 PPT 等。
- **SVG**：矢量图，适合论文排版，可无损缩放，字体嵌入更稳定。

### 6.3 控制台中文输出（可选）

若终端编码导致 `print` 中文报 `UnicodeEncodeError`，可使用安全输出：

```python
def safe_print(message: str) -> None:
    try:
        print(message)
    except UnicodeEncodeError:
        print(message.encode("ascii", errors="backslashreplace").decode("ascii"))
```

## 七、快速排查清单

遇到“中文变方框”时，按以下顺序检查：

| 序号 | 检查项 | 操作 |
|------|--------|------|
| 1 | 当前字体链 | 打印 `plt.rcParams["font.sans-serif"]`，确认是否包含 SimHei、Microsoft YaHei 等 |
| 2 | 本机可用字体 | `from matplotlib import font_manager; print([f.name for f in font_manager.fontManager.ttflist if "Hei" in f.name or "YaHei" in f.name])` |
| 3 | Seaborn 覆盖 | 若用了 `sns.set_theme()`，确保**在其之后**再次设置 `font.sans-serif` |
| 4 | 输出格式 | 尝试保存为 SVG，对比 PNG 是否仍有方框 |
| 5 | 字体安装 | Windows 可安装「微软雅黑」「黑体」等，确保系统中有至少一种中文字体 |
