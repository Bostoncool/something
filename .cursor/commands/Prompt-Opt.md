# 提示词规范化指令

请将下列用户输入转化为**规范化、书面化**的提示词，并直接输出优化后的版本供用户使用或发送。

## 转化规则

1. **术语规范化**
   - 将口语化、俚语或非正式表述替换为对应的学术/科技专有名词
   - 保留并正确使用领域术语（如：机器学习、深度学习、数据可视化等）
   - 若有歧义或模糊表述，采用业界通用、定义明确的术语

2. **语言精简**
   - 删除冗余、重复或无关表述
   - 使用简洁句式，避免冗长从句
   - 在保证语义完整的前提下，优先选用更精炼的词汇

3. **结构清晰**
   - 若涉及多步操作或多项要求，按逻辑顺序分点列出
   - 明确区分：目标、约束、输入/输出、期望行为

## 输出格式

- 直接给出优化后的提示词文本
- 若存在关键术语替换，可在文末用「术语对照」简要说明（可选）
